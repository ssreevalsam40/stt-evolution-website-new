# STT Evolution Website

A premium single-page website that chronicles the history and evolution of Speech-to-Text (STT) transcription technology.

## Features
- **Visual Timeline**: A journey through key milestones from 1952 to today.
- **Bento Grid**: A technical deep dive into how modern ASR engines work.
- **Premium Design**: Built using the VoicePulse design system with smooth animations and high-quality visuals.

## Built With
- Semantic HTML5
- Vanilla CSS
- JavaScript & Intersection Observer API
- Custom Al-generated assets

## Hosting Locally
You can host this locally using any static file server:
```bash
# Using Python
python -m http.server 8000
```
Then visit `http://localhost:8000`.
